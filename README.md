## anywordle back-end

### 简述

* 该项目为anywordle后端服务，采用Koa（typescript）+typeorm操纵mysql
* 功能：
  * 登录
    * 用户主动登录
      * 方式
        * 需输入邮箱和密码
        * 第三方登录 => 接入google第三方登录
      * 登录后右上角登录字样变为用户相关字样
    * 用户被动登录（使用导出功能触发登录）
      * 方式
        * 与主动登录同理
      * 登录后右上角登录字样变为用户相关字样，且自动触发导出（即用户之前的导出意图得到实现）
  * 注册
    * 需输入字段
      * 邮箱
      * 密码（两次，其中一次为确认）
      * 验证码（通过特定邮箱发送到用户邮箱）
